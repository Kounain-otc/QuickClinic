# 🏥 QuickClinic

A modern **Doctor Appointment Booking Web App** built using React.js and Tailwind CSS that allows users to find doctors, book appointments, and manage schedules easily.


 Live Demo
 https://quickclinic-frontend.onrender.com  

 GitHub Repository
 https://github.com/Kounain-otc/QuickClinic  

 ---

 ##Features

-  User Authentication (Login / Signup)
-  Browse Doctors by Specialization
-  Detailed Doctor Profiles
-  Smart Appointment Booking System
-  Real-time Slot Availability
-  Cancel / Manage Appointments
-  Fully Responsive UI
-  Smooth Navigation with React Router
-  State Management using Context API

---

## Responsive Design

QuickClinic is built with a **mobile-first approach**:

- Works seamlessly on mobile, tablet, and desktop
- Adaptive layouts using Tailwind CSS
- Optimized navigation for small screens
- Touch-friendly UI components
